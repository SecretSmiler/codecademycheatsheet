Cheatsheet: HTML/CSS

Semantic HTML: Giving meaning, improving accessibility, SEO

Semantic structure of a web page

Tag Name Description
<header> Header Information about the page, e.g. title
<nav> Navigation Navigation links. Often placed within <header> or <footer>
<main> Main Main body of content
<section> Section Thematically linked aspects of a page, for example, Gaming section
<article> Article Stand-alone content, using above example, a game review.
Other examples: blogs, newsletters
<aside> Aside Content that is seperate to the main theme, e.g. advertisements. Enhancement of theme, usually seperate to the main body of text.
<figure> Figure An image
<figcaption> Figure Caption Caption to describe the image, child to <figure> - caption stays linked to the image.
<footer> Footer E.g. copyright info.

Media Tags

Tag Name Description
<video> Video Used to link video content.
<audio> Audio Used to link audio content
<embed>* Embed Embeds media such as browser plugins.
E.g. <video source src="video.mp4" type='video/mp4'> File Not Supported </video>

Media Attributes

Tag Attribute
<video> src, type, height, width, controls, loop, autoplay, muted, title
<audio>
<embed>* src, type, height, width, title
